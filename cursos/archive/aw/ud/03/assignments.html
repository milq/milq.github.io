3. Servicios de gestión de archivos web

Crea un manual sobre cómo instalar, configurar y hacer funcionar un servicio de alojamiento de archivos multiplataforma en la nube como OwnCloud.

Nota: antes de proceder a la instalación hay que activar sudo para nuestro usuario: enable sudo on an user account on Debian.

Para ello, descargamos XAMPP para Linux (la versión de PHP 7.1.), luego descargamos y descomprimimos como usuario (nunca como root) owncloud. Luego ejecutamos sudo mv owncloud /opt/lampp/htdocs. Luego en /opt/lampp/htdocs/owncloud ejecutamos mkdir data y después chmod 777 data/.

Iniciamos XAMPP (sudo /opt/lampp/xampp start), entramos en phpMyAdmin desde el navegador (localhost/phpmyadmin) y creamos una base de datos y, si hiciera falta, un usuario con todos los privilegios. Instalamos OwnCloud (localhost/owncloud), ejecutamos y nos dirá que hay que cambiar a 0770. Para resolverlo, ejecutamos en /opt/lampp/htdocs los siguientes comandos: sudo chown daemon:daemon -R data/ y luego sudo chmod 770 -R data/. Por último, ejecutamos OwnCloud y debería funcionar. Para más ayuda pulsa aquí.

El manual será un documento XHTML (owncloud.html) que tendrá que estar validado en XHTML 5.2 y WCAG 2.0 AAA y con estilo validado en CSS3.

El manual contendrá los siguientes apartados:

  1. Establece la utilidad de un servicio de gestión de archivos web.
  2. Describe diferentes aplicaciones de gestión de archivos web además de OwnCloud.
  3. Instala y adapta una herramienta de gestión de archivos web como OwnCloud.
  4. Crea y clasifica cuentas de usuario en función de sus permisos.
  5. Gestiona archivos y directorios.
  6. Utiliza archivos de información adicional.
  7. Aplica criterios de indexación sobre los archivos y directorios.
  8. Comprueba la seguridad del gestor de archivos.
