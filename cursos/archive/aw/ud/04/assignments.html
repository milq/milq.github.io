4. Aplicaciones de ofimática web

Crea un documento sobre cómo utilizar dos aplicaciones de ofimática web: Google Docs y Office 365.

El documento será un documento XHTML (docs_office.html) que tendrá que estar validado en XHTML 5.2 y WCAG 2.0 AAA y con estilo validado en CSS3.

El documento contendrá los siguientes apartados:

  1. Establece la utilidad de las aplicaciones de ofimática web Google Docs y Office 365.
  2. Describe las diferentes aplicaciones de ofimática web (procesador de textos, hoja de cálculo, entre otras).
  3. Usa estas aplicaciones de ofimática web sin conexión a internet.
  4. Gestiona las cuentas de usuario de las diferentes aplicaciones web ofimáticas.
  5. Aplica criterios de seguridad en el acceso de los usuarios.
  6. Reconoce las prestaciones específicas de cada una de las aplicaciones instaladas.
  7. Utiliza las aplicaciones de forma colaborativa.

Entrega

  • Desde la carpeta de usuario en Dropbox, y dentro de la carpeta UT, crea una carpeta 04.
  • Dentro de la carpeta 04 sube el manual docs_office.html (respetando las minúsculas y la extensión) antes de las 23.59 del jueves 15 de febrero de 2018.
