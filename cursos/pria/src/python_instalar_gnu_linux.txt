# Cómo instalar Python en GNU/Linux

## Instalación en distribuciones basadas en Debian (Ubuntu, Linux Mint, etc.)

1. Abre la terminal.
2. Actualiza la lista de paquetes con `sudo apt-get update`
3. Instala Python con `sudo apt-get install python3`
4. Verifica la instalación con `python3 --version`

## Instalación en distribuciones basadas en RPM (Fedora, CentOS, RHEL, etc.)

1. Abre la terminal.
2. Actualiza la lista de paquetes con `sudo dnf update`
3. Instala Python con `sudo dnf install python3`
4. Verifica la instalación con `python3 --version`
