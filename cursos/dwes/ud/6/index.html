6. Aplicaciones web asíncronas

Actividades

1. Asincronía en las aplicaciones web

1. Realiza los siguientes ejercicios:
a) Explica qué es la asincronía, la E/S asíncrona y describe estos dos ejemplos.
b) Define qué es una aplicación web asíncrona y busca, al menos, tres ejemplos reales.
c) Lee, investiga y explica con tus propias palabras lo que es Ajax, para qué sirve y mira este vídeo.
d) Explica este ejemplo Ajax que consulta y muestra datos en una base de datos de manera asíncrona.
e) Lee, investiga y explica con tus propias palabras lo que es un SPA.
f) Lee, investiga y explica con tus propias palabras lo que es un full-stack web developer.

2. Node.js, Npm y Laravel Breeze

2. Realiza los siguientes ejercicios:
a) Lee, investiga y explica con tus propias palabras lo que es Node.js.
b) Lee, investiga y explica con tus propias palabras lo que es npm y mira este vídeo.
c) Descarga e instala la versión estable LTS de Node.js.
d) Comprueba que funciona Node.js y Npm en tu terminal ejecutando node -v y npm -v.
e) Explica qué es Laravel Breeze e instala y ejecuta una aplicación web Laravel Breeze de ejemplo.
f) Haz una demo con el ejemplo anterior de Laravel Breeze registrando un usuario e iniciando sesión.

3. Librerías y frameworks JavaScript

3. Realiza los siguientes ejercicios:
a) Lee, investiga y explica con tus propias palabras lo que es una librería JavaScript.
b) Lee, investiga y explica con tus propias palabras lo que es un framework JavaScript.
c) Di en qué se diferencia una librería JS de un framework JS y nombra tres ejemplos de cada uno.
d) Lee, investiga y explica con tus propias palabras lo que es una React.
e) Lee, investiga y explica con tus propias palabras lo que es un Angular.
f) Lee, investiga y explica con tus propias palabras lo que es un Vue.js.

5. Aprende Vue.js

Entra en la página oficial de Vue y visualiza el vídeo Why Vue.js.

El procedimiento a realizar en cada lección es el siguiente:

  3. Resuelve el reto (challenge).
