7. Crea el controlador

Después de crear el modelo Libro, migrarlo y actualizarlo, crea el controlador.

Laravel facilita la creación de controladores y adición de métodos estilo CRUD.

A continuación, ejecuta:

php artisan make:controller LibroController --resource --model=Libro

Este comando creará el controlador LibroController y de forma predeterminada siete métodos como el indexar, almacenar, crear, actualizar, destruir, mostrar, editar.

Puedes ver el resultado en app/Http/Controllers/LibroController.php. Automáticamente, como puedes observar, se ha creado la clase LibroController que deriva de la clase Controller y que contiene métodos para realizar operaciones CRUD con el modelo Libro.

Ahora necesitamos proporcionar implementaciones para estos métodos.
