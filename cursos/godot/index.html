Instalación en Godot C# en GNU/Linux

  1. Instala .NET 6 SDK y VS Code. Luego en Code instala las extensiones C# y C# Tools for Godot.
  2. Si usas GNU/Linux, instala Mono y en Code: ⚙→Settings→Omnisharp: Use Global Mono→always.
  3. Instala Godot C# y en Godot C#: Editor→Editor Settings→Mono→Editor→External editor→VS Code.
  4. En Code comprueba IntelliSense y depura con Run and Debug→Create a launch.json file→C# Godot.
  5. Más información aquí.
  6. Crea un acceso directo Godot.desktop en ~/.local/share/applications/ y pon en Exec tu ruta de Godot.
