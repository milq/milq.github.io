Instalación en Godot C# en GNU/Linux

  1. Si usas GNU/Linux, descarga e instala Mono.
  2. Instala .NET 6 SDK y VS Code. Luego en VS Code instala las extensiones C# y C# Tools for Godot.
  3. Descarga e instala Godot C#.
  4. Editor→Mono→Editor→External editor→Visual Studio Code
